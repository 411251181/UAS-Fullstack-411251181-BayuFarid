Berikut adalah Soal UAS, silakan di kerjakan dulu yak

Berdasarkan implementasi dari pengembangan middleware serta backend services (API) sebelumnya ( Project Eco-Share ), anda diminta oleh perusahaan tersebut untuk mengembangkan frontend services ( web application dengan mobile-responsive ) yang mencakup transaksi dan autentikasi serta yang memiliki arsitektur sistem, sebagai berikut :
Arsitektur & Keamanan: Mengembangkan web aplikasi yang memiliki fitur autentikasi terintegrasi dengan API (middleware) login dan otorisasi yang membedakan hak akses antara penyewa dan pemilik barang.
Integrasi Data & Bisnis: Mengembangkan modul transaksional yang sudah terimplementasi logika peminjaman yang kompleks (misal: pengecekan ketersediaan stok secara otomatis dan pencatatan riwayat transaksi) berdasarkan API yang sebelumnya.
Reliabilitas: Menampilkan keseragaman (unified) terhadap penanganan error agar user mudah memahami terkait issue yang terjadi.

Fitur Utama :
Implementasi fitur Login termasuk Role Access sesuai dengan API.
Implementasi modul serta fitur pada scope Transaksional sesuai dengan modul pada API yang dikembangkan sebelumnya.
Manajemen Environment Variables dan Git Workflow (minimal 3 commit dengan pesan yang standar)

Ketentuan Soal :
Framework yang digunakan adalah VueJS
Database yang digunakan MySQL ( berdasarkan project sebelumnya “Eco-Share” )
Repository menggunakan Git ( gitlab / github ) ( bisa menggunakan repository yang sama / berbeda )
Pastikan dilakukan testing dan review sebelum di submit ke KULON
Pastikan semua kebutuhan teknis yang diperlukan untuk membangun aplikasi dengan secara mandiri dilakukan dengan menganalisis arsitektur yang akan dikembangkan.
